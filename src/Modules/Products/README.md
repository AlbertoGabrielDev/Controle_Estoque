# Products Module (Piloto)

Modulo piloto para iniciar a modularizacao na Fase 2.

Estrutura criada nesta fase:
- `Http/Controllers`
- `Http/Requests`
- `Services`
- `Repositories`
- `Models`
- `Database/Migrations`
- `Database/Seeders`
- `Database/Factories`
- `Jobs`
- `Console/Commands`
- `Events`
- `Listeners`
- `Resources/js/Pages`
- `Resources/views`
- `Routes`
- `Tests`

Nesta fase ainda nao houve migracao de codigo de producao para este modulo.
