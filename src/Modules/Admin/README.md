# Admin Module

Modulo administrativo iniciado na Fase 6.

Status inicial desta etapa:
- `Usuarios`: Controller, Requests, Service e Model migrados para o modulo.
- `Roles/Permissoes/Menus`: Controller, Requests, Service, Repository/RepositoryEloquent e Models migrados para o modulo.
- Rotas web e paginas Vue de `Users` e `Roles` co-localizadas no modulo.
- DB artifacts de `Usuarios`, `Roles`, `Permissoes` e `Menus` (`migrations`, `seeders`) migrados para `Database/*`.
- Wrappers de compatibilidade permanecem em `App/*`, `database/*` e `resources/js/Pages/*`.
